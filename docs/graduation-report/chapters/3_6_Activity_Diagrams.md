# 3.6 مخططات النشاط (Activity Diagrams)

يوضح مخطط النشاط التدفق المنطقي للإجراءات والقرارات داخل النظام. سنركز في هذا القسم على أعقد مسار عمل في المنصة وهو **"عملية نقل ملكية مركبة"**، والتي توضح التكامل بين الفحوصات التلقائية والتدخل البشري.

## مخطط نشاط نقل ملكية مركبة

يوضح المخطط التالي تسلسل الخطوات من لحظة بدء البائع للعملية وحتى اعتمادها نهائياً:

```mermaid
activityDiagram
    start
    :البائع يبدأ عملية النقل;
    :إدخال الرقم الوطني للمشتري;
    if (هل توجد ديون أو مخالفات؟) then (نعم)
        :إظهار رسالة خطأ (المعاملة محظورة);
        :توليد فاتورة بالديون للمواطن;
        stop
    else (لا)
        :إرسال إشعار لحظي للمشتري;
        :انتظار قرار المشتري;
        if (هل قبل المشتري العرض؟) then (لا)
            :إلغاء المعاملة وإخطار البائع;
            stop
        else (نعم)
            :المشتري يحول الثمن عبر البنك;
            :إدخال رقم المرجع البنكي;
            if (هل رقم المرجع صحيح؟) then (لا)
                :تنبيه المشتري لتصحيح الرقم;
                backward
            else (نعم)
                :تحويل المعاملة للموظف الحكومي;
                :الموظف يراجع الوثائق القانونية;
                if (هل الوثائق سليمة؟) then (نعم)
                    :تحديث قاعدة البيانات (نقل الملكية);
                    :إرسال إشعارات النجاح للطرفين;
                    stop
                else (لا)
                    :رفض المعاملة مع ذكر السبب;
                    stop
                endif
            endif
        endif
    endif
```

## تحليل المسار
1.  **نقاط القرار التلقائية:** يقوم النظام بفحص الديون بشكل آلي في البداية، مما يمنع هدر وقت الموظف في معاملات غير قانونية.
2.  **التفاعل اللحظي:** يظهر المخطط كيف ينتظر النظام استجابة المشتري، وهو ما يتم تقنياً عبر WebSockets لضمان السرعة.
3.  **التكامل المالي:** لا تصل المعاملة للموظف إلا بعد تأكيد النظام البنكي لصحة الحوالة، مما يضمن الشفافية المالية.
4.  **الاعتماد البشري:** يمثل الموظف الخطوة النهائية للتأكد من الجوانب القانونية التي قد لا تغطيها الأتمتة الكاملة.

يعكس هذا المخطط قدرة النظام على إدارة "آلة حالة" معقدة تضمن حقوق كافة الأطراف والالتزام بالقوانين النافذة.
