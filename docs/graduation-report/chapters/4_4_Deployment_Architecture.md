# 4.4 معمارية النشر والبنية التحتية (Deployment Architecture)

تعتمد **منصة الحكومة الإلكترونية** على بنية تحتية موزعة تضمن استقرار النظام وسرعة وصول المستخدمين إليه. يوضح هذا الفصل كيفية نشر المكونات المختلفة والتقنيات المستخدمة في الربط بينها.

## 1. بنية النظام الموزعة (Infrastructure Design)
يتكون النظام من ثلاث وحدات أساسية يتم نشرها بشكل متكامل:
*   **خادم الواجهة الأمامية (Frontend PWA):** يتم نشره كـ "تطبيق ويب تقدمي" (Progressive Web App) باستخدام Vite، مما يسمح للمواطنين بتثبيته على هواتفهم كأنه تطبيق أصيل (Native App).
*   **خادم الواجهة الخلفية (Backend API):** يعتمد على بيئة Node.js ويتم نشره خلف وكيل عكسي (Reverse Proxy) لإدارة حركة المرور وتأمين الاتصالات.
*   **محرك البيانات (PostgreSQL):** يتم استضافته كخادم قاعدة بيانات مستقل لضمان أمن البيانات وفصلها عن منطق البرمجة.

## 2. ميزات تطبيق الويب التقدمي (PWA Features)
تم تصميم الواجهة الأمامية لتكون PWA لتوفير الميزات التالية:
*   **التثبيت على الهاتف:** يمكن للمواطنين إضافة أيقونة المنصة إلى شاشة الهاتف الرئيسية دون الحاجة لمتجر تطبيقات.
*   **استراتيجية التخزين المؤقت (Caching):** يستخدم النظام "Service Workers" لتخزين الملفات الأساسية، مما يسرع من عملية التحميل ويسمح بفتح الواجهات الأساسية حتى في حالات ضعف الاتصال.
*   **تجربة المستخدم السلسة:** يوفر الـ PWA مظهر واجهة مستخدم يشبه التطبيقات الأصلية، مع دعم كامل للإشعارات اللحظية.

## 3. الربط والاتصال (Network Connectivity)
*   **بوابة الـ Socket.io:** يتم تخصيص منفذ (Port) خاص لإدارة اتصالات WebSockets الدائمة، مما يضمن تدفق التنبيهات اللحظية دون تداخل مع طلبات الـ HTTP التقليدية.
*   **بروتوكول الأمان:** يتم فرض تشفير TLS/SSL لكافة الاتصالات لضمان خصوصية البيانات الوطنية الحساسة المتبادلة.

## 4. القابلية للتوسع الأفقي (Horizontal Scalability)
بفضل المعمارية المستخدمة، يمكن توسيع النظام مستقبلاً عبر:
*   زيادة عدد مثيلات (Instances) الخادم الخلفي لمواجهة زيادة عدد الطلبات.
*   استخدام موازنات الحمل (Load Balancers) لتوزيع حركة المرور بين الخوادم المتاحة.
*   توسيع سعة قاعدة البيانات عبر تقنيات النسخ الاحتياطي والقراءة الموزعة (Read Replicas).

تضمن هذه المعمارية أن تكون المنصة جاهزة للانتقال من مرحلة المشروع الجامعي إلى مرحلة التشغيل الفعلي على نطاق وطني واسع.
