# الشريحة 9: أصحاب المصلحة في النظام (Stakeholders)

## توزيع الأدوار والصلاحيات (RBAC)
تم تصميم النظام لخدمة أربع فئات رئيسية، لكل منها واجهة وصلاحيات برمجية مستقلة:

*   **المواطن (Citizen):** المستفيد الأول؛ يمتلك صلاحية الوصول لسجلاته المدنية، ممتلكاته (مركبات/عقارات)، وتسديد ذممه المالية.
*   **الموظف الحكومي (Employee):** يمثل جهة التدقيق؛ يمتلك لوحة تحكم لمراجعة طلبات الولادة والزواج واعتماد عمليات نقل الملكية النهائية.
*   **البنك المركزي/التجاري:** يمثل الضامن المالي؛ يتحقق من الحوالات المالية المرتبطة بالخدمات الحكومية عبر أرقام مرجعية فريدة.
*   **مدير النظام (Admin):** يشرف على استقرار المنصة، إدارة الحسابات، ومراقبة تدفق البيانات بين القطاعات.
