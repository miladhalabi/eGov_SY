# الشريحة 10: المتطلبات الوظيفية - الجزء الأول

## قطاع الشؤون المدنية (النفوس)
*   **إخراج القيد الرقمي:** توليد مستندات رسمية (فردي/عائلي) بصيغة PDF قابلة للتحقق عبر الـ QR.
*   **تسجيل الواقعات:** مسار عمل متكامل لتسجيل الولادات والزواج يشمل رفع المستندات الثبوتية واعتماد الموظف.

## قطاع المرور والنقل
*   **لوحة تحكم المركبات:** عرض تفصيلي لكافة السيارات المسجلة برقم المواطن الوطني.
*   **نقل الملكية اللحظي:** إدارة "المصافحة الرقمية" بين البائع والمشتري مع فحص آلي للمخالفات المرورية قبل البدء.
