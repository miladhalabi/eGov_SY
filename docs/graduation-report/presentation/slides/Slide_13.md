# الشريحة 13: معمارية وهيكلية النظام (Architecture)

## بنية المستودع الموحد (Monorepo Approach)
تم تنظيم المشروع برمجياً ككتلة واحدة تحتوي على:
*   **الخادم (Backend):** محرك Node.js مع Express لإدارة الـ APIs والـ WebSockets.
*   **الواجهة (Frontend):** تطبيق React (Vite) مبني كـ PWA لضمان السرعة والتجاوب.

## ميزات هذه المعمارية
1.  **سرعة المزامنة:** سهولة مشاركة تعريفات البيانات بين الواجهة والخلفية.
2.  **النشر الموحد:** تبسيط عمليات الاختبار الشامل (End-to-End Testing).
3.  **النمطية (Modularity):** تقسيم الخدمات إلى مجالات (Civil, Traffic, Tax) يسهل تطوير كل منها بشكل مستقل دون التأثير على استقرار النظام ككل.
