# الشريحة 16: المعمارية الأمنية (Security)

## حماية البيانات والعمليات السيادية
نظراً لحساسية النظام، تم تطبيق ثلاثة مستويات من الحماية:

*   **المستوى 1 (التوثيق):** استخدام **JWT** لإدارة جلسات الدخول، مع تشفير كلمات المرور باستخدام **Bcrypt**.
*   **المستوى 2 (الأمان المالي):** تطبيق نظام **Bank PIN**؛ لا يمكن للمواطن شراء سيارة أو دفع ضرائب بمجرد الدخول لحسابه، بل يجب إدخال رمز سري بنكي مستقل، مما يحمي أصوله حتى لو تم اختراق كلمة سر المنصة.
*   **المستوى 3 (التراخيص):** نظام **RBAC** (التحكم المستند للأدوار) يمنع أي مستخدم من الوصول لبيانات غيره على مستوى الـ API.

## التشفير في النقل
يتم فرض بروتوكولات آمنة لتشفير البيانات المتبادلة بين المتصفح والخادم، مما يحمي خصوصية المواطن ضد هجمات التسمع.
