# الشريحة 17: التنفيذ التقني - الواجهة الخلفية (Backend)

## بناء القلب النابض للنظام
تم تطوير الخادم باستخدام **Node.js** و **Express** لضمان التعامل الكفء مع الطلبات المتعددة.

## الميزات البرمجية المنفذة
*   **محرك القواعد (Rules Engine):** تطبيق المنطق القانوني (مثل التحقق من العمر عند تسجيل الولادة أو التحقق من الملكية عند البيع).
*   **إدارة المرفقات:** نظام لرفع وتخزين الوثائق الثبوتية بشكل آمن مع ربطها بالسجلات المدنية.
*   **نظام الإشعارات:** بناء خدمة لإرسال التنبيهات سواء عبر الـ REST API أو عبر الـ Sockets لضمان وصول المعلومة للمواطن فوراً.
