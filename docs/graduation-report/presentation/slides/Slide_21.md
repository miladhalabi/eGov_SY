# الشريحة 21: نتائج النظام - بوابة المواطن (Citizen Portal)

## واجهة واحدة لكافة الخدمات
تتيح بوابة المواطن تجربة متكاملة تشمل:
*   **لوحة التحكم الرئيسية:** تعرض ملخصاً للحالة المدنية، عدد المركبات، وإجمالي الديون المالية.
*   **مركز الطلبات:** لمتابعة حالة المعاملات (قيد المراجعة، مقبولة، بانتظار الدفع).
*   **صندوق الوثائق:** لاستعراض وتحميل كافة المستندات الرقمية التي تم استخراجها مسبقاً.

## سهولة الوصول
التصميم المتجاوب يسمح للمواطن بإنجاز معاملاته عبر الهاتف المحمول أثناء تنقله، مما يقلل من الازدحام في الدوائر الحكومية.
